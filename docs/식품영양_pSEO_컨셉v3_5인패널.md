## 1. v2 → v3 변경점

1. **페르소나 전략 전환**: YMYL 콘텐츠 가상 작성자(김데이터·이건강 등) 폐기.
   - 프로그래매틱 페이지 = **무기명 사실 데이터** (출처: 식약처)
   - 허브/가이드(Layer 2~3) = **영양사 실명 감수** (E-E-A-T 신뢰 신호)
2. **법률 제약의 기능화**: 식약처 영양강조표시 기준 충족 여부를 데이터로 판정.
   - 표현: "식약처 *저열량* 기준 100g당 40kcal 미만 / 이 제품 측정값 35kcal → 충족"
   - **금지**: 마케팅 배지화, CPA 링크 인접 강조, 우리 명의의 강조표시
3. **지표 다축화 + 정규화 표준**: 단일 효율 랭킹 → 다축. 비교 기준 = **100g / 100kcal**.
4. **CPA 정제**: 건강기능식품 제외(광고 사전심의·규제), 일반 고AOV 가공식품 + lead-gen.
5. **게이트 추가**: K8(영양강조표시 판정 필드 채움률).

## 2. 법정 영양강조표시 기준 (기능 구현 근거 — 2026.1.1 시행 반영)

> 출처: 식약처 「식품등의 표시기준」. 사이트는 **영업자의 표시가 아니라
> 제3자의 객관 비교 정보**로만 제시 (기준 명시 + 측정값 + 충족여부).

| 강조표시 | 법정 기준 (확인분) |
|----------|---------------------|
| 저열량 | 100g당 40kcal 미만 (또는 100ml당 20kcal 미만) |
| 열량 "무" | 100ml당 4kcal 미만 |
| 당류 "저" | 100g당 5g 미만 (또는 100ml당 2.5g 미만) |
| 무당 | 100g(ml)당 0.5g 미만 |
| 무가당 | 당류·기능적 대체원료·당 함유 원재료 미사용 + 효소분해 등 당 증가 없음 |
| **2026.1.1 개정** | 무당·무가당·제로슈거 강조 시 **감미료 함유 + 열량 정보 병기** 의무 |

> ⚠️ 위 표는 검색 확인분이며, 빌드 전 「식품등의 표시기준」 최신 별지 원문으로
> 고단백·고식이섬유·저나트륨 등 **전체 기준 재확인 필수** (법률 정밀화 = 5인→빌드 단계).

## 3. 컨셉 v3 요약

- **제품 정체성**: 무기명 사실 기반 **파생지표 데이터셋 + 목적 기반 랭킹**.
- **파생/교차 지표 (다축)**:
  - 100kcal당 단백질·당류·나트륨·포화지방
  - 같은 카테고리 내 **백분위** (모집단 = 동질 카테고리)
  - **식약처 영양강조표시 기준 충족 판정** (저열량·당류저·무당 등, 객관 비교 형식)
  - 대안 추천 ("같은 칼로리·더 높은 단백질")
- **페이지 계층**:
  - 주력 인덱싱 = 목적/카테고리 **랭킹 허브** (영양사 실명 감수)
  - 개별 제품 = 다축 지표 + 백분위 + 기준 판정 + 대안 (무기명, noindex 후보)
- **표현 가드레일**: 효능·질병 표현 금지 / 정성 폄하 금지 / 강조라벨은 기준+측정값 동반.

## 6. Gate 0 / Gate 0.5 (갱신)

### Gate 0 — 수요+경쟁+제휴 (반나절, 투자 0)
- 0a DataLab: 목적/랭킹형 키워드 그룹 C 중심 (단백질 많은 편의점 음식, 저당 과자, 다이어트 간식 추천 등) + 조회형 대조군 A
- 0b SERP/AI Overview 수동 캡처: AI Overview 점유·강자 점유·랭킹도구 결과 유무
- 0c 제휴 적격성: 쿠팡 일반 고AOV 가공식품 요율 + 다이어트/밀키트 lead-gen 제휴처 스캔
- 판정: K1·K2·K7

### Gate 0.5 — API 필드 (≈25콜 프로브)
| 측정 | Kill |
|------|:--:|
| 업체명·품목제조보고번호 채움률 | K3 |
| 에너지/탄단지 기준량(100g vs 1회제공량) 판별·혼재 | K4 |
| 가공식품 vs 원재료 vs 음식 분포 | K5 |
| 분류체계 균일성·결측 (백분위 모집단) | K6 |
| **열량·당류·나트륨·단백질 + 기준량 채움률** | **K8** |
| 식품코드 vs 품목제조보고번호 유일성 | (dedup) |

> 단위·기준 추정 코딩 금지 (연봉지도 minSal 교훈).
