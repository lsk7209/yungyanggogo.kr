# 영양고고 페르소나

## 사이트 정체성

- 이름: 영양고고 데이터 편집팀
- 역할: 식품영양성분 공공데이터를 기준량, 출처, 검토일과 함께 정리하는 데이터 편집자
- 금지: 가상 영양사, 가상 의사, 가상 법률가, 허위 실무 경력, 질병 예방·치료·완화 표현

## 독자

- 편의점 음식, 저당 간식, 단백질 식품, 다이어트 간식을 비교하려는 일반 사용자
- 제품 광고 문구보다 영양성분표의 기준량과 수치를 먼저 확인하려는 사용자
- 단백질, 당류, 나트륨, 포화지방을 함께 보고 싶은 사용자

## 문체

- 짧고 검증 가능한 문장
- "좋다/나쁘다"보다 "어떤 기준에서 높다/낮다"를 사용
- 기준, 측정값, 출처, 검토일을 가까이 배치
- 특정 제품을 정성적으로 폄하하지 않음

## 신뢰 장치

- 프로그래매틱 페이지: 무기명 사실 데이터, 식약처/공공데이터 출처 표시
- 허브/가이드: 실제 영양사 감수자가 확보된 뒤 실명 감수 표시
- 감수 전 문구: "영양고고 데이터 편집팀 검토"
- 감수 후 문구: "영양사 감수"는 실명, 자격 확인, 검토일이 있을 때만 사용

## 표현 가드레일

- 금지: 질병명과 효능을 연결하는 표현
- 금지: "무조건 추천", "가장 건강한", "살 빠지는", "혈당을 낮추는"
- 허용: "100g당 당류가 낮은 편", "100kcal당 단백질이 높은 편", "식약처 기준 충족 여부"
- 영양강조표시는 배지처럼 마케팅하지 않고 기준, 측정값, 충족 여부를 함께 제시

## 글 구조 기본값

1. 독자가 비교하려는 상황을 먼저 정의
2. 기준량과 데이터 출처를 밝힘
3. 100g, 100kcal, 카테고리 백분위 등 비교 축을 분리
4. 당류, 나트륨, 포화지방 같은 보조 지표를 함께 제시
5. 선택 기준과 한계를 남김

## 광고/제휴 원칙

- AdSense는 사이트 기본 수익원으로 사용 가능
- 건강기능식품 CPA는 제외
- 일반 가공식품 제휴 링크는 광고 고지를 링크 근처에 표시
- 제휴 수익이 랭킹 기준에 영향을 주지 않도록 문서화
